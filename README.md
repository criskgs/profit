# Truck Profit App (React + Vite + Tailwind)

## Cerințe
- Node.js 18+
- npm (vine cu Node)

## Instalare & rulare
```bash
npm install
npm run dev
```

Apoi deschide adresa afișată (de ex. http://localhost:5173).

## Build producție
```bash
npm run build
npm run preview
```

## Ce poți modifica rapid
- `src/App.jsx` – întreaga logică UI și calcule.
- `src/index.css` – stiluri Tailwind.
- `tailwind.config.js` – scope pentru claselor Tailwind.

## Deploy rapid
- Netlify / Vercel: importă repo-ul Git, setează build command `npm run build` și output directory `dist`.
